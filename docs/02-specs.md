# CLI Todoアプリ 仕様書

## 概要

ターミナル上でタスクの管理（追加、一覧表示、完了）を行うためのシンプルなCLIアプリケーション。

## コマンド体系

### 1. タスクの追加 (`add`)

新しいタスクをTodoリストに追加します。

- **使い方**: `haskell-todo add "<タスクの内容>"`
- **実行例**: `haskell-todo add "牛乳を買う"`
- **期待する動作**:
  - タスクが新しくリストに追加されます。
  - 追加に成功した旨のメッセージ（例: `Added: 牛乳を買う`）が表示されます。

### 2. タスクの一覧表示 (`list`)

現在登録されている未完了のタスクを一覧表示します。

- **使い方**: `haskell-todo list`
- **実行例**: `haskell-todo list`
- **期待する動作**:
  - 登録されているタスクがリスト形式で出力されます。
  - 操作（完了など）を行えるよう、各タスクには連番（ID）が振られて表示されます。
  - 出力イメージ:

    ```text
    1: 牛乳を買う
    2: 部屋の掃除をする
    ```

  - タスクが一つもない場合は、「タスクはありません」などのメッセージを表示します。

### 3. タスクの完了 (`done`)

指定した番号のタスクを完了状態（リストから削除）にします。

- **使い方**: `haskell-todo done <タスク番号>`
- **実行例**: `haskell-todo done 1`
- **期待する動作**:
  - 指定した番号のタスクがリストから取り除かれます。
  - 完了した旨のメッセージ（例: `Done: 1`）が表示されます。
  - 存在しない番号を指定した場合は、エラーメッセージを表示して終了します。

## データ保存先（非機能要件）

- タスクのデータは、実行ディレクトリ内のテキストファイルやJSONファイルなどに保存され、コマンドを実行するたびに状態が保持・復元されるものとします。
